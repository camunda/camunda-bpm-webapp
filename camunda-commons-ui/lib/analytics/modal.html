Camunda Needs You

Enable Telemetry Data

At Camunda we want to collect telemetry for the purpose of enhancing your experience. Data helps Camunda understand how the product is being used, which parts of the engine need improvement and which features we should implement next.

Telemetry also helps our team better understand the reasons why you use Camunda. With this knowledge we are able to deliver more value to our community and to you.

By enabling telemetry you can:

  • Contribute to the wider community
  • Help Camunda improve the BPM engine

What do we collect and why?

  • Meta and usage data

    Prouduct version, database technology, application server, installation ID.

    To better understand the environment on which our product is being used.

Privacy Preferences

To enhance user experience, Camunda BPM engine can integrate with Camunda Services, which requires external network requests. Please choose from the setting below.

Allow Camunda BPM to send anonymous usage statistics. We use this information to provide you a stable and improved product experience in the environment you are using. This allows Camunda to collect information about the product version and technical environment you are using, and how you are using it.

With any of these options, none of your personal information will be submitted.
